Guide

This is the moderation dashboard. The Activity tab is a live, permanent record of every staff action and every user identity change, so the team stays accountable to each other.

Each card is one event. The colored left edge and icon tell you the category. Developer names are red, moderator names are orange. Click any underlined name to trace that user: you will see every name they have used and every action taken against them.

Raw IP addresses are visible to developers only. Moderators can still ban or IP-block a user, since the server handles the address for them, but never see it.

Type under any entry to leave a permanent note or ask a question. All staff see comments. The Ban list and Moderators tabs are developer-only.
